# Conway’s Game of Life

A minimal, interactive implementation of Conway’s Game of Life using vanilla HTML, CSS, and JavaScript.

## Demo
https://YOUR_USERNAME.github.io/game-of-life/

## What it is

This is a simple cellular automaton where each cell follows three rules:

- A live cell with 2 or 3 neighbours survives
- A dead cell with exactly 3 neighbours becomes alive
- Otherwise, cells die or remain dead

Despite these simple local rules, complex global patterns emerge.

## Controls

- Start / Pause simulation
- Clear grid
- Randomise grid
- Click cells to toggle state

## Purpose

This project is intended as an educational visualisation to demonstrate how simple local interactions can generate complex emergent behaviour.

It is also a conceptual stepping stone toward more advanced simulation methods such as lattice-based fluid models.
