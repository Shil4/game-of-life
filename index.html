Conway's Game of Life

Rules

  • Any live cell with 2 or 3 neighbours survives
  • Any dead cell with exactly 3 neighbours becomes alive
  • All other cells die or remain dead

How to use

  • Click on the grid to turn cells on or off
  • Press Start to run the simulation
  • Press Pause to stop it at any time
  • Use Clear to reset the grid
  • Use Random to generate a random pattern

Each cell evolves based only on its neighbours. Watch how simple local interactions produce complex patterns.
